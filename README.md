## À quoi sert ce mod ?

**Essential CosmetiK Pro** est un mod client pour **Essential Mod** (Fabric). Il permet de :

- Porter **tous les cosmétiques Essential** (capes, ailes, accessoires…) sans achat.
- **Voir les outfits** des autres joueurs qui utilisent aussi le mod.
- Jouer en **LAN**, sur un **serveur classique** ou en **World Share Essential** — aucun plugin serveur requis.

La synchronisation des tenues ne transite **pas** par l'infrastructure officielle d'Essential. Les clients moddés échangent leurs outfits via un **relais dédié** (hébergé séparément), ce qui permet la sync entre amis sans dépendre des serveurs Essential.

> Un joueur **sans** le mod ne verra pas tes cosmétiques débloqués — comportement normal d'Essential.

## Installateur portable

Ce dépôt distribue **CosmetiK Installer** : un petit logiciel Windows qui installe le mod en un clic.

| Fonction | Détail |
|--------|--------|
| Détection auto | Vanilla, NoRisk, Prism, CurseForge, Modrinth, MultiMC… |
| Recherche | Filtre tes profils par nom ou version Minecraft |
| Portable | Aucune installation — lance l'`.exe` et c'est parti |

### Installation rapide

1. Télécharge **`CosmetikInstaller-*-portable.exe`** dans [Releases](https://github.com/ErrorNoName/Cosmetik-Pro/releases/latest).
2. Lance l'installateur.
3. Sélectionne ton profil Minecraft (ex. Fabric 26.1.2).
4. Clique sur **Installer le mod**.
5. Lance Minecraft avec **Essential Mod** déjà présent sur le profil.

---

## Prérequis

- **Minecraft** avec **Essential Mod** (Fabric)
- **Windows** pour l'installateur portable
- Jouer avec au moins un ami qui a aussi le mod pour la sync des outfits
